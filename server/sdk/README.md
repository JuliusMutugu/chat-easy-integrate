# Nego SDK – Embeddable Chat Widget

Add a Nego chat widget to your website so visitors can chat directly with your team.

## Quick Start

1. In Nego, open a room and click **Invite**
2. Scroll to **Embed widget** and click **Create embed code**
3. Copy the generated script tag
4. Paste it into your website's HTML before `</body>`

## Embed Code

```html
<script
  src="https://yourserver.com/sdk/nego.js"
  data-nego-token="YOUR_WIDGET_TOKEN"
  data-nego-url="https://yourserver.com"
></script>
```

## Parameters

| Attribute        | Required | Description                                  |
|------------------|----------|----------------------------------------------|
| `data-nego-token`| Yes      | Widget token (from Nego room → Invite → Embed) |
| `data-nego-url`  | Yes      | Your Nego server URL (e.g. https://app.nego.com) |

## Behavior

- Loads a chat bubble in the bottom-right corner of the page
- Visitors chat as "Visitor-xxx" (anonymous)
- Messages are delivered to the linked Nego room in real time
- No account required for visitors

## API

- `GET /api/widget/init?token=xxx` – public init (returns roomId, roomName)
- `POST /api/widget/create` – create widget (auth required)
- `GET /api/widget/list/:roomId` – list widgets for room (auth required)
- `DELETE /api/widget/:id` – delete widget (auth required)

## Security

- Widget tokens are unique per room
- Tokens can be regenerated by creating a new widget
- Set `allowedOrigins` to restrict which domains can use the widget (future)
